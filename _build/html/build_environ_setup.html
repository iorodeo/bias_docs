Setting up a build environment for BIAS on Windows¶

In this section we describe how to set up a build environment for BIAS. on Window (7 & 8).

  • Setting up a build environment for BIAS on Windows

Setting up a build environment on Windows consists of the following steps:

  1. Download and install gcc
  2. Download and install Active State Perl (Required for Qt build)
  3. Download, Configure and Build Qt
  4. Download Configure, and build OpenCV
  5. Install mercurial
  6. Download and build the latest version of BIAS

Note, it is important that the correct versions gcc, Qt and OpenCV are used. We go through each of these steps in more detail below.
